# TestAutomation-E2EProject
Test Automation project using Selenium, Maven, TestNG, Jenkins, Extent Reports, Page Objects, Log4j. The tests of this project are simple to check titles, navigation bar and login. The main idea here is show how many tools and concepts we can use in the same project.
